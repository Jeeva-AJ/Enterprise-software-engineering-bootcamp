## 🧠 Reflection

### 1. What responsibilities do enterprise engineers have?

Enterprise engineers are responsible for designing, developing, and maintaining large-scale software systems used by businesses or organizations. Their responsibilities include:

- Building scalable and reliable backend systems
- Designing system architecture
- Writing clean and maintainable code
- Collaborating with teams using Git and code reviews
- Ensuring application security and performance
- Integrating APIs and external services
- Managing deployments and CI/CD pipelines
- Monitoring and debugging production systems

---

### 2. What types of systems do they build?

Enterprise engineers typically build:

- Web applications (e.g., e-commerce platforms)
- REST APIs and microservices
- Banking and financial systems
- Enterprise Resource Planning (ERP) systems
- Customer Relationship Management (CRM) systems
- Cloud-based applications
- Data processing systems
- Real-time systems (chat apps, notifications)
