# Enterprise Software Engineering Bootcamp

## 📌 Project Overview
This repository documents my 100-day journey to becoming proficient in enterprise-level software engineering.

The goal is to build real-world skills used in modern software companies, including backend development, system design, DevOps practices, and scalable application architecture.

---

## 🎯 Learning Goals
- Understand enterprise system architecture
- Master backend development (Node.js / APIs)
- Learn frontend integration basics
- Work with databases and data modeling
- Use Git and GitHub professionally
- Understand CI/CD pipelines
- Learn Docker and containerization
- Build scalable and maintainable applications

---

## 🧰 Tech Stack
- Node.js
- Express.js (planned)
- Docker
- Git & GitHub
- VS Code
- Postman / Thunder Client
